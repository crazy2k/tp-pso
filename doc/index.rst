Trabajo Práctico de Programación de Sistemas Operativos
=======================================================

Este proyecto es un *Trabajo Práctico* realizado en el marco de la
`materia Programación de Sistemas Operativos
<http://dc.uba.ar/materias/progso/2011/index_html>`_, dictada en el
`Departamento de Computación <http://dc.uba.ar/>`_ de la `Facultad de
Ciencias Exactas y Naturales <http://exactas.uba.ar/>`_ de la
`Universidad de Buenos Aires <http://www.uba.ar/>`_.

Se trata de un sistema operativo sencillo, escrito con el solo
propósito de aprender y experimentar. El sistema fue y es desarrollado
por `Pablo Antonio <https://github.com/crazy2k>`_ y `Pablo Herrero
<https://github.com/pabloh>`_. El `repositorio git del proyecto
<https://github.com/crazy2k/tp-pso>`_ se encuentra alojado en `GitHub
<https://github.com/>`_.
